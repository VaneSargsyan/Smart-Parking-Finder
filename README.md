 Smart Parking Finder

Smart Parking Finder is a web application that helps users quickly find and reserve available parking spots through an interactive map. It reduces time spent searching for parking, minimizes traffic congestion, and improves the overall driving experience in busy cities.

---

Features

-  Real-time parking map (Green = free, Red = occupied)
-  Reservation timer for temporary holds
-  Favorites system for preferred spots
-  Dark mode for night use
-  Multilingual support (English, Armenian, Russian)
-  Firebase authentication (Login/Register)

---

 Goal

To make parking faster, easier, and more efficient in urban areas by providing a smart, user-friendly platform.
